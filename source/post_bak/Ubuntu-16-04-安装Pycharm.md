# 在线安装 #

添加源：

    $ sudo add-apt-repository ppa:mystic-mirage/pycharm

安装免费的社区版：

	$ sudo apt update
	$ sudo apt install pycharm-community

安装收费的专业版：

	$ sudo apt update
	$ sudo apt install pycharm

事实上，这种方法简单方便，不需要事先安装JDK环境等，一切都会自动安装完成。
